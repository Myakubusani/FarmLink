# FarmLink
Connecting Farmers Directly with Buyers
# 🌾 FarmLink

FarmLink is a web application designed to connect farmers directly with buyers, making it easier to discover, sell, and purchase agricultural products.

## 🚀 Live Demo

FarmLink is deployed using Firebase Hosting:

https://farmlink-29241.web.app

## 🎯 Problem

Many farmers struggle to connect directly with buyers, which can lead to limited market access, dependence on middlemen, and difficulty selling produce quickly.

FarmLink provides a simple digital marketplace where farmers can list their agricultural products and buyers can discover available produce.

## ✨ Features

- 🌍 Multiple language options
- 📱 Simple phone number and PIN authentication
- 🌾 Farmers can post agricultural products
- 🔍 Buyers can browse and search for produce
- 📍 Product location information
- ❤️ Add products to My Favorites
- 🛒 Place and manage orders
- ⭐ Product ratings
- ✏️ Edit product listings
- 🗑️ Delete products and orders
- 📞 Contact farmers directly
- 📱 Responsive design for mobile and desktop
